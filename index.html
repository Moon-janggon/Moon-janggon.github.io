프로젝트 1: 스마트 오피스 AIoT 환경 모니터링

기간: 2024.09 ~ 2025.06 NHN Academy AIoT 2기 부트캠프 | 팀 프로젝트

요약: Spring 및 IoT 센서로 연결된 서버를 모니터링하는 APM 서비스입니다.

기술 스택: Java, Spring Boot, MSA, MySQL/InfluxDB, Docker

  • 주요 기능 및 당신의 역할:
    • IoT 센서 데이터 수집 및 처리 모듈 개발: Spring Boot 기반의 RESTful API 엔드포인트 설계 및 구현. Spring WebFlux와 Reactor 라이브러리를 활용하여 고성능 데이터 수신 모듈 개발.
    • 데이터 파이프라인 구축 및 저장소 관리: InfluxDB를 메인 시계열 데이터베이스로 연동하여 대용량 시계열 데이터의 고속 쓰기 및 범위 쿼리 최적화. MySQL에 관계형 데이터 스키마 설계 및 관리.
    • API 설계 및 배포 담당: 마이크로서비스 아키텍처 원칙에 따른 서비스 간 명확한 책임 분리 및 RESTful API 명세 정의. Docker 컨테이너 이미지 빌드 및 GitHub Actions를 활용한 CI/CD 워크플로우 구축.
  • 문제 해결 및 배운 점:
    • 대규모 IoT 센서 데이터 및 서버 리소스의 안정적인 수집 및 결측치 처리: 비동기 처리와 시계열 데이터베이스(InfluxDB) 특성 이해 및 적용. 불완전한 센서 데이터 결측치 보정 로직 구현을 통한 데이터 신뢰성 확보.
    • 마이크로서비스 아키텍처 설계 및 원격 서버 배포 역량 확보: MSA 설계 과정에서 서비스 역할과 경계 정의의 중요성 학습. Docker 컨테이너화와 GitHub Actions를 통한 CI/CD 파이프라인 구축 경험으로 실무 역량 향상.
